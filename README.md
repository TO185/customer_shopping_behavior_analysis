# 🛍️ Customer Shopping Behavior Analysis - Python + SQL + Power BI

## 🎯 Project Overview
Analyzed customer shopping behavior from 3,900 transactions across multiple product categories to uncover spending patterns, customer segments, and discount impact.

## 📈 Key Metrics

| Metric | Value |
|:---|:---|
| Total Transactions | 3,900 |
| Total Sales | $2.27M |
| Total Profit | $282,858 |
| Unique Customers | 793 |

## 🔍 Analysis Features

| Feature | Insight |
|:---|:---|
| Customer Segmentation | New (15%) vs Returning (45%) vs Loyal (40%) |
| Revenue by Category | Technology (45%) vs Furniture (32%) vs Office (23%) |
| Monthly Trend | November-December sales spike 35-50% |
| Discount Impact | 20%+ discounts = $100 loss per order |
| Subscription Impact | Subscribers spend 35% more than non-subscribers |
| Top Customers | Tamara Chand: $8,964 profit |

## 💡 Business Recommendations

1. **Boost subscriptions** - Subscribers spend 35% more
2. **Limit discounts to 15%** - 20%+ discounts cause losses
3. **Loyalty program** - Top 10% customers drive 45% revenue
4. **Increase inventory before November** - Holiday season spike
5. **Target age 26-35** - Contributes 40% of revenue
